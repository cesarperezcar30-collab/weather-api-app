# 🌤️ Weather API App

A responsive weather application built with **HTML**, **CSS**, and **JavaScript** that allows users to search for weather conditions in cities around the world using the Open-Meteo API.

> *(Add a screenshot of your app here later.)*

---

## ✨ Features

- 🔍 Search weather by city
- 🌡️ View current temperature in Fahrenheit
- 💨 Display wind speed in mph
- ☁️ Show current weather conditions
- 📅 5-day weather forecast
- 📍 Get weather using your current location
- ⌨️ Search by pressing the **Enter** key
- 🌙 Modern dark theme
- 📱 Responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo Weather API
- Open-Meteo Geocoding API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/cesarperezcar30-collab/weather-api-app.git
```

### Open the project

Open the folder in **Visual Studio Code**.

### Run the application

Install the **Live Server** extension in VS Code.

Right-click on `index.html` and select:

```
Open with Live Server
```
